    Summary

    Because data connections are often spotty, this delivers a timely weather report to the user through simlple, low data SMS protocols.

    While the client side is handled by the Twilio API, the server parses requests using regular expressions, fetches recent METARs from the Aviation Weather Center Text Data Server, and delivers raw METAR text for any requested K*** airport code.
