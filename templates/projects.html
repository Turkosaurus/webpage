InCanon

Datbase notebook for RPGs (web app)

Summary

The web app uses a relational database to allow users to collaboratively build a compendium of all the people, places, and things that animate a role-playing campaign setting.

Players and Game Masters can join their own shared campaign in which everyone views the index of notable entries as their home page, and anyone can add new information.

Originally, this app was a project submission to CS50x, but through continued development it has been released as a standalone web app.

Features

  • User management
  • Hashed passwords and input sanitization
  • Persistent user and game data
  • Scales to support multiple campaigns

Technologies Used

  • Python
  • Flask
  • Bootstrap
  • Jinja
  • SQLite3
  • PostgereSQL
  • Heroku

Nailivc Studios

Inventory and production management (web app)

Summary

Nailivic Studios creates folk art in a variety of sizes, colors, and shapes, all cut from computer controlled lasers.

This web app is being built to streamline inventory management and production tracking.

In addition, connection to the Shopify REST API will facilitate a dashboard for shipping needs.

Features

  • CSV import tool
  • item and part inventory management
  • needed parts are calculated from projected items

Technologies Used

  • Python
  • Flask
  • PostgereSQL
  • Bootstrap
  • Heroku
